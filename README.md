#  Sign Language Detection

A simple AI-powered web application that recognizes basic sign language gestures in real time using a webcam and a custom image classification model built with Google Teachable Machine.

## 📌 Overview

Communication is a fundamental human need, and sign language plays a vital role in enabling communication for individuals with hearing or speech impairments.

This project demonstrates how Artificial Intelligence can be used to recognize sign language gestures through a webcam and provide real-time predictions. While this is a basic implementation, it highlights the potential of AI in building more accessible and inclusive technologies.

## ⚙️ How It Works

1. A custom image classification model is trained using Google Teachable Machine.
2. The model is exported and integrated into the web application.
3. The user grants webcam access.
4. Video frames are captured in real time.
5. The AI model analyzes the hand gesture.
6. Predictions are displayed instantly on the screen.
